
# Типи та характеристики SSD-накопичувачів

## 1. Загальна характеристика SSD

**SSD (Solid State Drive)** — це твердотільний накопичувач даних, який використовує мікросхеми флеш-пам’яті для зберігання інформації замість механічних компонентів, як у HDD.

### Основні переваги SSD:

- Висока швидкість читання та запису
    
- Відсутність механічних частин
    
- Низьке енергоспоживання
    
- Безшумна робота
    
- Стійкість до вібрацій

## 2. Принцип роботи SSD

SSD складається з:

- Контролера (SSD Controller)
    
- Чіпів NAND Flash
    
- Буферної пам’яті (DRAM або DRAM-less архітектура)
    
- Інтерфейсу підключення
    

Контролер відповідає за:

- Керування записом і читанням
    
- Розподіл зносу (Wear Leveling)
    
- Корекцію помилок (ECC)
    
- Підтримку TRIM

# 3. Типи SSD за інтерфейсом підключення

## 3.1 SATA SSD

Працюють через інтерфейс SATA III (6 Гбіт/с).

### Характеристики:

- Максимальна швидкість читання: ~550 МБ/с
    
- Максимальна швидкість запису: ~500 МБ/с
    
- Форм-фактор: 2.5”
    

Переваги:

- Сумісність зі старими системами
    
- Доступна ціна
    

Недоліки:

- Обмеження швидкості через SATA
    

[МІСЦЕ ДЛЯ ЗОБРАЖЕННЯ: 2.5" SATA SSD]

---

## 3.2 mSATA SSD

Компактний варіант SATA SSD, що використовувався в ноутбуках.

Характеристики:

- Інтерфейс SATA
    
- Менший форм-фактор
    
- Швидкість аналогічна SATA
    

[МІСЦЕ ДЛЯ ЗОБРАЖЕННЯ: mSATA модуль]

---

## 3.3 M.2 SATA SSD

Фізично встановлюється в слот M.2, але працює через SATA.

Особливості:

- Компактний форм-фактор
    
- Обмеження швидкості SATA (~550 МБ/с)
    

[МІСЦЕ ДЛЯ ЗОБРАЖЕННЯ: M.2 SATA SSD]

---

## 3.4 M.2 NVMe SSD (PCIe)

Найсучасніший тип SSD, що використовує інтерфейс PCI Express та протокол NVMe.

### Характеристики:

- PCIe 3.0: до ~3500 МБ/с
    
- PCIe 4.0: до ~7000 МБ/с
    
- PCIe 5.0: понад 10 000 МБ/с
    

Переваги:

- Дуже висока пропускна здатність
    
- Низькі затримки
    
- Підходить для ігор та професійних задач
    

[МІСЦЕ ДЛЯ ЗОБРАЖЕННЯ: M.2 NVMe SSD у слоті на материнській платі]

## 3.5 PCIe SSD (карта розширення)

Виконується у вигляді плати для слота PCIe.

Особливості:

- Використовується у серверах
    
- Підтримка великої кількості ліній PCIe
    
- Висока продуктивність

# 4. Типи пам’яті NAND

## 4.1 SLC (Single-Level Cell)

- 1 біт на комірку
    
- Найвища надійність
    
- Висока вартість
    

## 4.2 MLC (Multi-Level Cell)

- 2 біти на комірку
    
- Баланс ціни та ресурсу
    

## 4.3 TLC (Triple-Level Cell)

- 3 біти на комірку
    
- Найпоширеніший тип
    
- Оптимальне співвідношення ціни та ресурсу
    

## 4.4 QLC (Quad-Level Cell)

- 4 біти на комірку
    
- Велика ємність
    
- Менший ресурс перезапису

# 5. Основні технічні характеристики SSD

## 5.1 Швидкість читання та запису

Визначає продуктивність у роботі з файлами.

## 5.2 IOPS (Input/Output Operations Per Second)

Показник кількості операцій введення/виведення за секунду.

## 5.3 TBW (Total Bytes Written)

Ресурс накопичувача — обсяг даних, який можна записати за весь термін служби.

## 5.4 MTBF (Mean Time Between Failures)

Середній час напрацювання на відмову.

## 5.5 Обсяг пам’яті

Типові значення:

- 256 ГБ
    
- 512 ГБ
    
- 1 ТБ
    
- 2 ТБ і більше

# 6. Порівняння типів SSD

---

# 7. Висновок

SSD-накопичувачі значно перевершують HDD за швидкістю та надійністю.  
Вибір залежить від:

- Типу інтерфейсу материнської плати
    
- Потрібної продуктивності
    
- Бюджету
    
- Призначення системи
    

Для сучасних систем оптимальним вибором є **M.2 NVMe SSD**, тоді як для модернізації старих ПК достатньо SATA SSD.

# Скрипт для відображення технічної інформації про встановлений накопичувач

## 1. Призначення програми

Даний скрипт написаний мовою пакетних команд Windows (Batch, `.bat`) і призначений для отримання повної технічної інформації про:

- фізичні накопичувачі (HDD/SSD)
    
- розділи диска
    
- логічні томи
    
- обсяг вільного місця
    
- версію операційної системи
    
- тест швидкості читання та запису
    

Скрипт використовує системні утиліти Windows:  
`wmic`, `fsutil`, `systeminfo`, `winsat`.

---

# 2. Лістинг скрипта

```bat
@echo off
chcp 65001 > nul
title Complete Disk Information Script
color 0A

echo =============================================
echo           COMPLETE DISK INFORMATION
echo =============================================
echo.

echo ===== PHYSICAL DISKS =====
wmic diskdrive get Model,SerialNumber,InterfaceType,Size,Status
echo.

echo ===== DISK PARTITIONS =====
wmic partition get Name,Size,Type
echo.

echo ===== LOGICAL DISKS =====
wmic logicaldisk get DeviceID,FileSystem,VolumeName,Size,FreeSpace
echo.

echo ===== DISK SPACE DETAILS =====
for %%D in (C D E F G) do (
    if exist %%D:\ (
        echo --- Drive %%D: ---
        fsutil volume diskfree %%D:
        echo.
    )
)

echo ===== SYSTEM INFORMATION =====
systeminfo | findstr /C:"OS Name" /C:"OS Version" /C:"System Type"
echo.

echo ===== DISK READ SPEED TEST =====
winsat disk -seq -read -drive C
echo.

echo ===== DISK WRITE SPEED TEST =====
winsat disk -seq -write -drive C
echo.

echo =============================================
echo            TEST COMPLETED
echo =============================================
pause
```

# 3. Пояснення роботи скрипта

## 3.1 Початкові налаштування

```
@echo off
chcp 65001 > nul
title Complete Disk Information Script
color 0A
```

- `@echo off` — приховує виконання команд.
    
- `chcp 65001` — встановлює кодування UTF-8.
    
- `title` — задає назву вікна.
    
- `color 0A` — встановлює зелений текст на чорному фоні.
